# ThumbsAI — AI Image Browser

> **Designed by:** Zero &nbsp;|&nbsp; **Built by:** Jarvis &nbsp;|&nbsp; **Version:** 1.0

A fast, offline image browser for Windows built around AI-generated image collections.  
Generates and caches thumbnails, reads AI generation metadata embedded in PNG files,
and lets you browse, search, sort, and organize thousands of images without any cloud dependency.

---

## Features

- **Persistent thumbnail cache** — folders open instantly after the first scan (SQLite-backed)
- **AI metadata reader** — extracts prompt, seed, model, sampler, CFG scale, and steps from PNG files generated by A1111, ComfyUI, NovelAI, and InvokeAI
- **Metadata editor** — edit and save generation metadata back to both the database and the PNG file (A1111 format, atomic write)
- **Watched folders** — auto-scan on launch and on file changes; new images appear without any manual action
- **Favorites** — pin frequently used folders to a fast flat list that bypasses filesystem tree scanning
- **Folder color indicators** — see at a glance which folders have been scanned, which have thumbnails, and which are watched or favorited
- **Similarity sort** — visually reorder the grid by perceptual similarity to a selected image
- **Photoshop `.8bf` plugin support** — apply Photoshop-compatible filters without Photoshop (Redfield plugin collection included)
- **Image operations** — rotate, flip, resize, blur, sharpen, color balance, HSLC, denoise, and more
- **Batch processing** — batch pipeline and batch save-as across multiple selected images
- **Video thumbnail support** — extracts keyframes from MP4, MKV, AVI, MOV (requires FFmpeg)
- **ZIP archive browsing** — browse images inside ZIP archives in-place
- **Launch Apps** — configure external apps (Photoshop, A1111 img2img, upscaler, etc.) in the right-click menu
- **Drag-to-website** — drag PNG thumbnails directly onto sites like CivitAI; embedded metadata transfers with the file

## Installation

1. **Clone or download** this repository
2. Double-click **`INSTALL.bat`**
   - Checks for Python 3.11+ (installs via `winget` if missing)
   - Creates a `.venv` virtual environment
   - Installs all required packages
3. Double-click **`run.bat`** to launch

> The app opens with no console window. A desktop shortcut to `run.bat` works the same way.

---

## Quick Start

1. Launch with `run.bat`
2. In the **Folder Panel** on the left, expand the drive tree and click a folder with images
3. ThumbsAI scans the folder in the background — thumbnails appear in real-time
4. Right-click a folder → **Always Watch this Folder** to keep it current automatically
5. Right-click any thumbnail → **Metadata** to view or edit AI generation parameters
6. Double-click a thumbnail to open the full-resolution viewer

|-----------|-----------|

| ComfyUI | `prompt` chunk (JSON node graph) |
| NovelAI | `Description` + `Comment` chunks |
| InvokeAI | `sd-metadata` chunk |

## User Data

All user-specific data is stored in the `data\` subfolder:

| File | Contents |
|------|----------|
| `data\thumbs.db` | Thumbnail cache and image metadata (SQLite) |
| `data\settings.json` | All settings — favorites list, watched folders, preferences |

Use the included scripts to manage backups:

```
BACKUP_DATA.bat   — timestamped snapshot of settings + database
RESTORE_DATA.bat  — restore from a previous snapshot
```

> **Updating?** Never overwrite the `data\` folder. Copy new source files over, run `INSTALL.bat`, launch.
